# TrajectoryFeatureGeneration
generate feature of trajectory.

## 主要步骤

1. 将每个用户的plt文件都合并为1个csv文件。
2. 生成停留点和移动点。
3. 删除异常值。
4. 生成指定数据格式的轨迹数据。
   1. 生成时序数据。
   2. 矩阵数据。

## 数据中需要注意的地方

1. 需要删除区域之外的点。
2. 生成停留点的时候，需要去掉停留点为空的用户。
3. Geolife的数据是按天提供的，需要删除停留点超过24小时的点。
4. 使用最直接最原始的方式对栅格进行编号。也就是按顺序进行编号。

## 其他

1. 只要不用特征附着，那么可以不用多线程和polars。
